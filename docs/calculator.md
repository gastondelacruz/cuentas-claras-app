# Transaction Calculator

Cuentas Claras includes an in-app calculator for personal income and expense transactions.

The calculator preserves the transaction draft while it is open, accepts the current amount as its initial value, and returns the calculated monetary result to the transaction form.

It supports arithmetic precedence, parentheses, percentages, decimals, deletion, clearing, and guarded invalid operations. The calculator is available from the personal transaction forms and remains outside bottom-tab navigation.
